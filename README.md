## Get started

1. Clone Repository
   ```bash
     git clone https://github.com/shalomihaza/bootcamp-buddy.git

      cd bootcamp-buddy
    ```
   

1. Install dependencies

   ```bash
   npm install
   ```

2. Prebuild Project

   ```bash
   npx expo prebuild 
   ```

3. Install Native Deps
   ```
   npx expo run:ios
   npx expo run:android
   ```

4. Start Project
   ```
   npx expo start
   ```
